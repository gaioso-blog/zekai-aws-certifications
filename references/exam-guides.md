# AWS Certification Exam Guides

Use the official AWS Certification Exam Guides page as the source of truth:
https://docs.aws.amazon.com/aws-certification/latest/examguides/aws-certification-exam-guides.html

The guide usually contains:
- target candidate description
- exam content outline
- domains
- task statements
- in-scope services
- out-of-scope services
- question format
- exam duration
- scoring information

When generating questions, map every question to a domain or task statement from the guide.

## Note on retired exams

- **MLS-C01 (AWS Certified Machine Learning - Specialty)** is retired and no longer available. Do not generate questions for this exam code.
- **AIP-C01** is **AWS Certified Generative AI Developer - Professional** — a Professional-level exam focused on designing, implementing, and deploying generative AI solutions on AWS. It is NOT a version of AI Practitioner (AIF-C01). Treat it as a fully available exam — no restrictions on domain-mapped question generation.
- **AIF-C01** is **AWS Certified AI Practitioner** — a Foundational-level exam. Do not confuse with AIP-C01.
- Do not invent domains, task statements, or percentages for any exam. Always use the official exam guide as the source of truth.
